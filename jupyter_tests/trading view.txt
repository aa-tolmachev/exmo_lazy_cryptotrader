По информации от Александра Кустов - он построил веб сервис, который автоматизирует торги только на основании информации о сигналах с сайта tradingview.

По ссылке - https://www.tradingview.com/crypto-screener/ показаны разные сигналы на покупку и продажу разных пар на разных биржах.

Настройки Александра Кустова - раз в 4 часа Стронг бай. Кажется, что там очень много факторов, которые нужно собирать и ими оперировать.

Для того чтобы увидеть метод, который предоставляет информацию нужно нажать на странице - cmd + alt + i тогда увидим приходящую информацию Scan. Там при нажатии указан урл с которого прилетает информация, там во входных данных json с информацией о сигналах, можно управлять фильтром для получения ценной информации.

Здесь к примеру - https://www.tradingview.com/symbols/ADABNB/technicals/ более детальная информация, которую тоже можно использовать
